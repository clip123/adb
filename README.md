# adb

bento PDF to run from windows
install Node.js

The simplest way to run is to download the dist.zip from the releases:
https://github.com/alam00000/bentopdf/releases/tag/v1.11.2

Do this:
1. Download dist-1.11.2.zip and unzip it
2. Open the folder
3. Right click and then select open in terminal
4. Type: npx http-server -c-1

This will spin up the server and you can access it at: http://localhost:8080/
